Anteriormente estaba usando Ubuntu como sistema operativo, pero ya hace un tiempo que utilizo windows con `WSL` y no la estoy pasando tan mal, es raro volver a Windows y decir eso - jajaja.

Por eso, ire compartiendo mis experiencias de desarrollo desde la ventana azul, y este sera el artículo donde se enlazarán una serie de artículos que escribiré, con la intención de volver a consultarlos cada vez que malogre mi configuración en `WSL` mientras exploro sus funcionalidades.

**¿Que debes esperar de esta serie de artículos?**

La idea es documentar mi proceso de setup para poder trabajar sobre Ruby on Rails y React, que son tecnologías que uso para el trabajo y para la mayoría de mis proyectos personales.

En esta serie de artículos también agregaré alguna que otra configuración necesaria para que todo funcione a tope, y ustedes podrán seguir el proceso de instalación, es por eso, que cada artículo procurará cubrir un tema a la vez, con la intención de que solo visitan el que necesiten en ese momento, y para facilitar su búsqueda sin tener que enredarnos tanto.

Espero les sea de ayuda, tanto como lo sera para mi.
