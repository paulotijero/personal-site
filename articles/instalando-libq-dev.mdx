Cuando intente ejecutar mis proyectos en `rails`, estos usan una gema llamada `pg` para conectar con la base de datos de `PostgreSQL`, el mensaje de ayuda nos dice que estemos seguros de estar usando la versión correcta de la gema, y cuando especificamos la versión `1.2.3` , el error persiste.

```bash
// Error de consola.

An error occurred while installing pg (1.2.3), and bundler cannot continue.
Make sure that `gem install pg -v '1.2.3' --source 'https://rubygems.org/'` succeeds before bundling.
```

Buscando en internet me tope con una solución que sí funcionó para mí, y solo me faltaba instalar una librería en mi [entorno de desarrollo](/articles/mi-setup-desarrollo-wsl-ubuntu)

```bash
$ sudo apt install libpq-dev
```
